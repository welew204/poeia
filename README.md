# Mixo.poeia: unlock the poetry of taste

This is a simple recipe app, designed around managing cocktail recipes (but can work for any procedure involving ingredients, wares (glasses, etc), and steps). This project uses a [Standard RedwoodSDK Starter template]([url](https://docs.rwsdk.com/tutorial/full-stack-app/setup/)) and a working demo is deployed at mixo.poeia.us (not working yet).
